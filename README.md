MH Accessories – E-Commerce Platform
MH Accessories is a full-stack e-commerce web application designed for selling digital tech accessories. This project was built to demonstrate proficiency in the MERN Stack and the application of Object-Oriented Programming (OOP) principles in a real-world scenario.

🚀 Features
User Authentication: Secure login and registration for customers.

Product Catalog: Dynamic display of tech accessories with filtering.

Shopping Cart: Add/remove functionality with real-time price calculation.

Admin Dashboard: (If applicable) Manage products and view orders.

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS / Bootstrap

Backend: Node.js, Express.js

Database: MongoDB

State Management: (e.g., Redux or React Context API)

Tools: Git, Postman (for API testing)

🏗️ Technical Highlights
OOP Principles: Utilized encapsulation and modular class structures in the backend to manage user and product data efficiently.

RESTful API: Developed a clean API architecture for seamless communication between the frontend and database.

Schema Design: Optimized MongoDB schemas for fast data retrieval and consistency.

⚙️ Installation & Setup
Clone the repository:

Bash
git clone https://github.com/mehtabhaidar1/MH-Accessories.git
Install dependencies for both Frontend and Backend:

Bash
npm install
Create a .env file and add your MongoDB URI.

Start the development server:

Bash
npm start
